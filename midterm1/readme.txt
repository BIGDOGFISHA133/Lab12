helpme.sh will have the user type in one of my chosen unix commands, then print out the man page for that command

midterm2.sh will take a text file I've chosen from Wikipedia and find the number of occurences of a keyword that the user types.

midterm3.sh will take files in a directory hierarchy that haven't been accessed in N days and compress them. N is a user input.

midterm4.sh acts as a phone book database, allowing the user to read, add, edit, or delete content

midterm5.c takes the factorial of a given number, and also can find the value of an integer bit-shifted left 3 times then added
to the one's complement of the original integer.
