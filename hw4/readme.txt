checkpasswd.c checks a given string, and uses a point-based system to see if the string is safe or unsafe as a password.

palindrome.c will check is a string is a palindrome.

swap.c will swap two variables without using a third one, and also swap all numbers in sentence1 with alphabets in sentence2
and vice versa.

Countrycode.c will have the user enter an international dialing code then looks it up in an array, displaying a country if it found one.
