sortnumeric.c takes a given array and sorts it in ascending or descending order depending on user input. It uses dynamic memory
allocation so any number of elements can be entered.

sortalphabetic.c takes a given array and sorts it in alphabetical order, ascending or descending based on user input.

covidarray.c allows the user to register users for a vaccine registration form, taking inputs of first name, last name, 
DOB, sex, dose number, date of previous dose, type of vaccine, and zipcode. The program then generates an 8 letter 
alphanumeric code unique to the user based on the previous inputs.
